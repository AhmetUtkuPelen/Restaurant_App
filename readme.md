# Restaurant Application

This is a mobile-friendly restaurant application built to manage online orders, reservations, and payments.

## Technology Stack

- **Front End**: TypeScript, React

[text](https://imgur.com/Ha6uiN6)
- **Back End**: Python, FastAPI

## Features

### General
- **Landing Page**: Provides an overview of the application features including online ordering, table reservations, and payment processing.
- **About Page**: Displays the restaurant's location and history.
- **Contact Page**: Shows contact information and includes a functional contact form for users to send emails.

### User Account
- **Authentication**: Users can register for a new account and log in.
- **Profile Management**: Authenticated users can edit their profile information.
- **User Dashboard**: Authenticated users can view their past orders, reservations, and favorite products.

### Products and Ordering
- **Product List**: Browse products by category. Includes functionality to search for items, add them to the cart, and mark them as favorites.
- **Product Details**: View detailed information for a single product. Users can leave comments and rate products.
- **Ordering**: Users can create orders containing items from their cart.
- **Payment**: A payment process is required to complete orders and reservations.

### Reservations
- **Table Booking**: Users can make reservations by selecting a date, time, number of guests, and a specific table.

### Administration
- **Back End Landing Page**: Contains redirect links for system health checks, Swagger UI API documentation, and the Admin Dashboard.
- **Admin Dashboard**: Accessible to users with the Admin role. Allows management (create, edit, delete) of application data models.
