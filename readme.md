# Restaurant Application

This is a mobile-friendly restaurant application built to manage online orders, reservations, and payments.

## Technology Stack

- **Front End**: TypeScript, React

  ![Typescript](https://i.imgur.com/HP8BO3f.png)

  ![React](https://i.imgur.com/Ha6uiN6.png)

  <br>

- **Back End**: Python, FastAPI

  ![Python](https://i.imgur.com/syp9c3K.png)

  ![FastApi](https://i.imgur.com/wTJaBu7.png)

### Products and Ordering
- **Product List**: Browse products by category. Includes functionality to search for items, add them to the cart, and mark them as favorites.

  ![ProductList](https://i.imgur.com/JjpqXFp.png)

  ![ProductListSearch](https://i.imgur.com/w8S4kYH.png)

  <br/>

- **Product Details**: View detailed information for a single product. Users can leave comments and rate products.

  ![ProductSingle](https://i.imgur.com/I0C6qwn.png)

  ![ProductSingle3](https://i.imgur.com/xmPYKzw.png)

- **Ordering**: Users can create orders containing items from their cart.

- **Payment**: A payment process is required to complete orders and reservations.
